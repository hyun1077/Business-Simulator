# Business Simulator

Main app path: `biz-manager-app`

This repository currently contains:

- `biz-manager-app`: the Next.js business simulator app
- `excel-crm`: reference materials not used for deployment

## Deploy target

Use Vercel and set the project Root Directory to:

`biz-manager-app`

## Local run

```bash
cd biz-manager-app
npm install
npm run dev
```
